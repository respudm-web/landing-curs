# Landing curs — AI-видео курс

## Что за проект

Лендинг курса по созданию AI-видео. Одностраничный сайт на чистом HTML/CSS/JS без фреймворков.

## Стек

- HTML / CSS / JavaScript — без фреймворков и сборщиков
- Запуск: `npx serve -l 3000` из папки проекта
- Конфиг VS Code / Claude: `.claude/launch.json` (уже настроен)

## Ключевые файлы

| Файл | Назначение |
|------|-----------|
| `index.html` | Разметка всей страницы |
| `styles.css` | Все стили (темы, анимации, секции) |
| `script.js` | Логика: переключение темы, интерактив |

## Папки

- `assets/` — изображения, shapes (3D-фигуры), видео (`hero.mp4`, `ИИ Видео.mp4`)
- `assets/shapes/` — SVG / PNG 3D-фигуры для декора секций
- `Файлы для сайта/` — исходные файлы от пользователя (не деплоятся в продакшн)

## Темы (dark / light)

- **dark** — тема по умолчанию
- **light** — активируется добавлением класса `html.light`
- Переключение: кнопка `.theme-toggle` в хедере
- Сохранение: `localStorage` (ключ `theme`)

### Логотипы по теме

| Файл | Используется при |
|------|-----------------|
| `assets/logo-black.png` | тёмная тема (`.logo-img-dark`) |
| `assets/logo.png` | светлая тема (`.logo-img-light`) |

Видимость управляется классами `.logo-img-dark` / `.logo-img-light` через CSS.

## 3D-фигуры (`.shape-3d`)

Фигуры перекрывают границы секций и обрезаются по краям. Чтобы скрыть артефакты обрезки используется `mask-image` — пересечение двух `linear-gradient` по периметру (intersect двух linear-gradient), которое создаёт маску-рамку с прозрачными краями.

## Переходы между секциями (`.section-seam-top` / `.section-seam-bottom`)

Для плавного перехода между светлой и тёмной секцией используются псевдоэлементы с тёмным overlay высотой **280px** и multi-stop градиентом (от непрозрачного к прозрачному).

### Важная заметка по архитектуре

`section-dark.section-seam-top::before` имеет `display: none` — cream-tint оверлей был убран, так как создавал видимый стык между секциями.
